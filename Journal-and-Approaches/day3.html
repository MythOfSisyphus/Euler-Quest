Problem 15: Lattice Paths

I studied the Wikipedia page on Lattice Paths and got to know that number of paths from $(0, 0)$ to $(n, m)$ is given by $$\binom{n + m}{n}.$$

So, we should put $n = m = 20$ to get the desired answer and then I wrote little python code for calculations.
